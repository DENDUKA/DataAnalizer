# BTC Price Change Analysis

## Описание

Проект для загрузки исторических данных криптовалют с Binance API и анализа изменений цены за различные временные периоды.

## Загруженные данные

| Файл | Пара | Таймфрейм | Период | Свечей |
|------|------|-----------|--------|--------|
| btcusdt_30m_4years.csv | BTC/USDT | 30 мин | 21.01.2022 - 21.01.2026 | 70,126 |
| ethusdt_30m_4years.csv | ETH/USDT | 30 мин | 21.01.2022 - 21.01.2026 | 70,126 |
| solusdt_30m_4years.csv | SOL/USDT | 30 мин | 21.01.2022 - 21.01.2026 | 70,126 |
| xrpusdt_30m_4years.csv | XRP/USDT | 30 мин | 21.01.2022 - 21.01.2026 | 70,126 |

**Часовой пояс:** UTC+4

### Структура CSV файлов

| Колонка | Описание |
|---------|----------|
| OpenTime | Время открытия свечи |
| Open | Цена открытия |
| High | Максимум |
| Low | Минимум |
| Close | Цена закрытия |
| Volume | Объём в базовой валюте |
| CloseTime | Время закрытия свечи |
| QuoteVolume | Объём в USDT |
| TradesCount | Количество сделок |

## Анализ изменений цены BTC

### Файлы анализа

| Файл | Описание | Период |
|------|----------|--------|
| btc_price_changes_analysis.xlsx | Анализ с диаграммами | 4 года |
| btc_price_changes_analysis_v2.xlsx | Анализ с диаграммами | Последний год |

### Методология

- Анализируется изменение цены закрытия (Close) за периоды от 1 до 24 часов
- Изменения группируются по интервалам $100
- Для каждого интервала рассчитывается вероятность (% от общего числа измерений)

### Результаты (4 года)

| Период | Измерений | Среднее изменение | Диапазон |
|--------|-----------|-------------------|----------|
| 1 ч | 70,124 | +1$ | -8,634$ .. +6,066$ |
| 6 ч | 70,114 | +9$ | -10,821$ .. +9,223$ |
| 12 ч | 70,102 | +17$ | -13,322$ .. +8,575$ |
| 24 ч | 70,078 | +36$ | -12,902$ .. +8,879$ |

### Распределение для 1 часа (топ интервалы)

| Изменение цены | Вероятность |
|----------------|-------------|
| -100$ .. 0$ | 25.06% |
| 0$ .. +100$ | 25.50% |
| -200$ .. -100$ | 10.03% |
| +100$ .. +200$ | 10.44% |

~50% часовых изменений укладываются в диапазон ±$100.
